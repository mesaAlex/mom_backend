API Documentation

Welcome to the Meals on Mesa REST API. Use the endpoints below to access recipe data for the React client application.

GET /api/recipes

Returns an array of all recipes, each containing an id, title, description, image path, tags, timing, servings, calories, ingredients, and instructions.

Example Response

[ { "id": 1, "title": "Salmon Salad", "description": "A fresh, high-protein salad …", "image": "/images/salmon_salad.png", "tags": ["High-protein", "Low-carb", "Gut-friendly"], "prepMinutes": 10, "cookMinutes": 12, "servings": 2, "calories": 420, "ingredients": ["2 Salmon Fillets", "…"], "instructions": ["Preheat the oven …", "…"] }, … ]
GET /api/recipes/:id

Returns a single recipe object by its numeric id. Returns a 404 JSON error if the recipe is not found.

Example Response

{ "id": 1, "title": "Salmon Salad", "description": "A fresh, high-protein salad …", "image": "/images/salmon_salad.png", "tags": ["High-protein", "Low-carb", "Gut-friendly"], "prepMinutes": 10, "cookMinutes": 12, "servings": 2, "calories": 420, "ingredients": ["…"], "instructions": ["…"] }
GET /images/:filename

Static file serving. All recipe images are available at /images/<filename> (e.g., /images/salmon_salad.png).
